# Sistema de Login e Lista de Tarefas 
Este projeto foi desenvolvido como parte dos meus estudos em Engenharia de Software, com o objetivo de aplicar na prática conceitos de lógica de programaçao, desenvolvimento web e organzação de código.
A aplicação permite que usuário realizem cadastro, login e gerenciem suas próprias tarefas de forma simples e funcional, ultilizado armazenamento local no navegador. 

## Funcionalidade 

- Cadastro de usuários 
- Login com validação de dados 
- Criação, edição e exclusão de tarefas 
- Listagem dinâmica de tarefas 
- Armazenamento com LocalStorage
- Interface simples e responsiva 
- Organização do código em funções 

## Tecnologias Utilizadas 

- HTML5
- CSS3
- JavaScript
- LocalStorage 
- Git e GitHub 

## Objetivo do Projeto 

O objetivo do projeto é consolidar o aprendizado em: 

- Lógica de programação 
- Manipulação do DOM 
- Estruturação de funções 
- Organização de arquivo 
- Boas praticas no desenvolvimento web 
- Simulação de um sistema real de usúario e tarefas
